== Requirements ==

* We always want to see where a link goes (internal or external)
* We always want to know where a link opens (same tab, new tab, new window)
* We want to allow the user to open the link where he wants to (same tab, new tab, new window)
* We want to force-open some links in specific modes (same tab, new tab, new window) when the user only clicks normally.
