== Information ==

The footer may contain the following links 

* Privacy-link
* About-link
* Disclaimer-link
* Imprint-link

More information: https://www.mediawiki.org/wiki/Manual:Footer
 
=== Privacy-Link ===
* [[MediaWiki:Privacy]] contains the text for the privacy-link.
* [[MediaWiki:Privacypage]] contains the name of the wiki page to which the privacy-link links to. It currently links to [[Project:Privacy]]

=== About-Link ===
* [[MediaWiki:Aboutsite]] contains the text of the about-link.
* [[MediaWiki:Aboutpage]] contains the name of the wiki page to which the about-link links to. It currently links to [[Project:About]]

=== Disclaimer-Link ===
* [[MediaWiki:Disclaimers]] contains the text of the disclaimer-link.
* [[MediaWiki:Disclaimerpage]] contains the name of the wiki page to which the disclaimer-link links to. it currently links to [[Project:Disclaimer]]
